# Sources Index

`sources/` 에 들어온 모든 원문의 메타데이터.

형식:
```
## <source-id>
- title: <원문 제목>
- type: article | paper | note
- url: <원문 URL>
- ingested: YYYY-MM-DD
- path: sources/articles/<file>
- summary: wiki/summaries/<slug>.md
```

> source-id 규칙: `<연도>-<짧은-슬러그>` (예: `2025-karpathy-llm-wiki`)
